3. **Deploy using Blueprint** (recommended):
   - Go to Render Dashboard â†’ New â†’ Blueprint
   - Connect your repository
   - Render will detect `render.yaml` automatically
   - Click "Apply"

4. **Set environment variables** in Render dashboard:
   - `GEMINI_API_KEY` - Your Gemini API key
   - `EMAIL_HOST_USER` - Your Gmail address
   - `EMAIL_HOST_PASSWORD` - Gmail App Password
   - `DEFAULT_FROM_EMAIL` - Your Gmail address

5. **Wait for deployment** (5-10 minutes)

6. **Run migrations**:
   - Go to Render Shell
   - Run: `python manage.py migrate`

7. **Create superuser** (optional):
   - Run: `python manage.py createsuperuser`

### Required (Set in Render Dashboard)
- `SECRET_KEY` - Auto-generated by Render
- `DEBUG` - Set to `False`

## ğŸ” Troubleshooting

### Build Fails
- Check `requirements.txt` includes all dependencies
- Verify Python version (3.11.0)
- Check build logs for errors

### Database Connection Error
- Verify `DATABASE_URL` is set correctly
- Use Internal Database URL (not external)
- Check database is running

### Static Files Not Loading
- Verify `collectstatic` ran successfully
- Check WhiteNoise is installed
- Verify static files paths

### ChromaDB Issues
- Verify paths are correct
- Check persistent disk is mounted
- Verify directories are created
